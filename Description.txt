Для работы программы должна быть создана БД MySQL с именем "ozlotto".
Подключение к БД через MySQL/ODBC Connector.
В программе для соединения с БД используется компонент TADOConnection.

Основная форма:
- панель управления: 
	- "UpDown1" - установка интервала опроса сайта
	- "StartBtn" - кнопка принудительного опроса сайта
	- "TimePanel" - индикатор времени оставшегося до следующего опроса
- "DGGrid1" - список обработаных лотерей 
- "DBGrid2" - результаты лотереи выбраной в "DGGrid1"
- "LogMemo" - ход выполнения программы

Алгоритм работы:

1. Подключение к БД "ozlotto". 
2. Если БД создана - подключение к таблице названий лотерей "lotto_names".
   Если таблица есть загружаем названия лотерей в список.  
   Если таблицы нет, то создаем с полями:
   	- id - int(10) AUTO_INCREMENT
   	- name - VARCHAR(50).
3. Загрузка сайта "http://www.ozlotteries.com/lotto-results#" во временный HTML-файл на диске.
4. Парсинг файла:
	- находим теги <div> содержащие информацию о последних 10-ти тиражах лотереи
	- проходим по каждому тегу, вичитываем информацию о тираже:
		- название лотереи;
		- дата проведения тиража;
		- номер тиража;
		- выиграшные номера;
		- имя победителя (для некоторых лотерей).
	- подключаемся к таблице с именем лотереи, если такой ещё нет создаем новую с полями:
		- Date - VARCHAR(20)
		- Draw - INT(10)
		- Results - VARCHAR(50)
		- Winner - VARCHAR(50)
	- сохраняем полученные данные в таблице.
5. Обновляем информацию из БД в таблицах основной формы.
6. Устанавливаем таймер следующего опроса сайта.
7. Для немедленного опроса жмем кнопку "Update Now".
